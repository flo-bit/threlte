This component is a port of [drei's `<Backdrop>`
component](https://github.com/pmndrs/drei#backdrop) a curved plane, like a studio backdrop. This is for presentational purposes, to break up light and shadows more interestingly.

<Example path="extras/backdrop" />

## Examples

### Basic Example

```svelte title="Billboard.svelte"
<script lang="ts">
  import { T } from '@threlte/core'
  import { Backdrop } from '@threlte/extras'
</script>

<Backdrop>
  <T.MeshStandardMaterial />
</Billboard>
```
